## bind()
:::info
bind() 方法主要用于修改函数的 this 指向
:::

bind() 会创建一个新的函数实例，并将该函数的 this 指向 bind() 方法的第一个参数

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>Document</title>
  </head>
  <body>
    <script>
      // bind() 方法会创建一个函数实例
      window.color = "red";
      function getColor() {
        console.log(this.color);
      }
      //这里的函数的调用方为 window
      getColor();

      const object = {
        color: 'blue',
      };
      //将
      const newGetColor = getColor.bind(object)
      newGetColor()
    </script>
  </body>
</html>

```
